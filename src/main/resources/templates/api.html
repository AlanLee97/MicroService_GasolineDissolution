接口说明

接口地址：
http://localhost:8080/rtnApi?T=293.15&P=1.0&deltag=0.6&delta0=0.8


请求方式：GET

参数：
参数名称 参数类型 是否必传 参数说明
T Double Y 温度，单位K，取值范围273.15~333.15
P Double Y 绝对压力，单位MPa，取值范围1.0~3.0
deltag Double Y 天然气对空气的相对密度，取值范围0.58~0.62
delta0 Double Y 脱气原油对水的相对密度，取值0.8

返回数据：
{
    "code":200,
    "msg":"ok",
    "data":{
        "note":"溶解油气比",
        "rs":4.8669238391386
    }
}
